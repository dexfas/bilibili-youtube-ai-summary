# 展示截图占位说明

把截图放到这个目录，并使用下面这些文件名，`docs/index.html` 会自动显示。

```text
showcase-browser-workbench.png       浏览器插件工作台
showcase-bilibili-page.png           B 站页面悬浮窗
showcase-task-progress.png           批量任务进度
showcase-workflow-menu.png           任务模式菜单
showcase-model-settings.png          Ollama 模型设置
showcase-youtube-scan.png            YouTube 扫描与标题翻译
showcase-youtube-title-translation.png YouTube 标题中文化结果
showcase-output-files.png            输出文件夹结果
showcase-output-folder.png           输出目录概览
showcase-transcript-preview.png      转写文本预览
showcase-cli.png                     命令行运行截图
```

如果暂时没有截图，网页会显示对应占位框。
